Android 版本

注意：本应用仅支持 Android 系统

本安装包为 Android APK 格式，不支持 iOS（苹果手机）/ 鸿蒙 / Windows / Mac 系统。

如何查看自己的手机系统？

  • 华为/荣耀：设置 → 关于手机 → Android版本
  • 小米/红米：设置 → 我的设备 → 全部参数 → Android版本
  • OPPO/Realme：设置 → 关于手机 → 版本信息 → Android版本
  • VIVO：设置 → 系统升级 → 右上角设置 → 关于手机
  • 三星：设置 → 关于手机 → 软件信息 → Android版本
  • 其他品牌：设置 → 系统/我的设备 → 关于手机 → Android版本

安装说明
  1. 点击上方"立即下载 APK"按钮下载安装包
  2. 打开手机设置 → 应用设置 → 应用管理
  3. 找到对应的浏览器或文件管理器
  4. 开启"允许安装未知来源应用"权限
  5. 返回下载目录，点击 APK 文件开始安装
