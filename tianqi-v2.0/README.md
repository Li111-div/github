# 🌤️ 超大型沉浸式天气 Web 应用

## ✅ 快速开始（3 步运行）

### 方法一：直接打开（最简单）

1. 找到项目文件夹中的 `index.html`
2. 双击文件，在默认浏览器中打开
3. 开始使用！✨

> **注意**：如果页面显示"缺少依赖"警告，请使用方法二（本地服务器）

---

### 方法二：使用本地服务器（推荐）

#### 选项 A：使用 Python

```bash
# 打开命令行/终端，进入项目目录
cd "d:\桌面\天气 v2.0"

# Python 3.x
python -m http.server 8080

# 或 Python 2.x
python -m SimpleHTTPServer 8080
```

然后在浏览器访问：**http://localhost:8080**

#### 选项 B：使用 Node.js

```bash
# 安装 http-server（只需一次）
npm install -g http-server

# 进入项目目录
cd "d:\桌面\天气 v2.0"

# 启动服务器
http-server -p 8080
```

然后在浏览器访问：**http://localhost:8080**

#### 选项 C：使用 VS Code

1. 用 VS Code 打开项目文件夹
2. 安装扩展："Live Server"
3. 右键点击 `index.html` → "Open with Live Server"
4. 自动在浏览器中打开

## 🔧 配置高德 API（可选）

当前项目使用**模拟数据**运行，如需真实天气数据：

1. 访问 https://lbs.amap.com/ 注册开发者账号
2. 创建应用获取 API Key
3. 打开 `js/common.js`，修改：
   ```javascript
   AMAP_API_KEY: '您的高德 API Key',
   ```
4. 详细说明请查看 `API 对接说明.md`

> ⚠️ **重要**：即使不配置 API，项目也能正常使用模拟数据！

## 🐛 常见问题

### Q: 页面空白/无法显示
**A**: 请使用本地服务器方式运行（见上方方法二）

### Q: 控制台显示警告
**A**: 如果是"缺少依赖"，检查 jQuery 和 ECharts 是否正确加载

### Q: 动画卡顿
**A**: 低端设备可减少背景粒子数量（修改 `js/weather.js`）

### Q: 数据不更新
**A**: 清除浏览器缓存或使用 Ctrl+F5 强制刷新
