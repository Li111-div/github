工具简介

密码加密解密工具箱是一款提供哈希加密与解码功能的在线工具，主要包含以下核心功能：

  • MD5 加密：将任意文本转换为 32 位 MD5 哈希值
  • SHA1 加密：将任意文本转换为 40 位 SHA1 哈希值
  • 哈希解密：通过密文查询数据库，返回原始明文
  • 离线工具：Base64、URL、进制转换等多种编码工具

所有加密操作会自动将明文和密文存储到云端数据库，方便后续通过密文反查明文。 离线工具的所有操作均在浏览器本地完成，不会向服务器上传任何数据。

MD5 加密功能

功能说明

MD5（Message-Digest Algorithm 5）是一种被广泛使用的密码散列函数， 可以将任意长度的数据转换为固定长度的 128 位哈希值（通常表示为 32 个十六进制字符）。

使用方法

  • 打开 MD5加密 页面
  • 在文本框中输入需要加密的明文
  • 点击「加密并存储」按钮
  • 系统将自动生成 MD5 哈希值并显示
  • 同时会将明文和密文自动存储到数据库

示例

输入：hello world
输出：5eb63bbbe01eeed093cb22bb8f5acdc3

SHA1 加密功能

功能说明

SHA1（Secure Hash Algorithm 1）是由美国国家安全局设计的安全哈希算法， 会产生一个 160 位的哈希值（通常表示为 40 个十六进制字符）。 SHA1 相比 MD5 产生更长的哈希值，理论上更安全。

使用方法

  • 打开 SHA1加密 页面
  • 在文本框中输入需要加密的明文
  • 点击「加密并存储」按钮
  • 系统将自动生成 SHA1 哈希值并显示
  • 同时会将明文和密文自动存储到数据库

示例

输入：hello world
输出：2aae6c35c94fcfb415dbe95f408b9ce91ee846ed

解密查询功能

功能说明

由于 MD5 和 SHA1 都是单向哈希算法，理论上无法逆向解密。 本工具的「解密」功能实际上是通过数据库查询实现的。 只有当明文曾经通过本工具的加密功能处理并存储到数据库时，才能查询到对应的明文。

使用方法

  • 打开 解密查询 页面
  • 选择与加密时相同的哈希算法（MD5 或 SHA1）
  • 输入需要查询的密文
  • 点击「查询解密」按钮
  • 如果数据库中有匹配记录，将显示对应的明文
重要提示

解密查询需要数据库中已有对应记录才能成功。如果数据库中没有该密文的记录， 则无法解密。因此，请确保加密和解密使用的是同一种算法。

离线编码工具

离线工具提供了多种常用的编码转换功能，所有操作均在浏览器本地完成， 不会向服务器上传任何数据。

Base64 编解码

Base64 是一种基于 64 个可打印字符来表示二进制数据的编码方式， 常用于在 HTTP 环境中传输二进制数据。

URL 编解码

URL 编码用于将特殊字符转换为 URL 安全格式， 例如空格转换为 %20。

进制转换

支持二进制、八进制、十进制、十六进制之间的相互转换， 输入时支持多种格式前缀：0x（十六进制）、0b（二进制）、0o（八进制）。

Unicode 编解码

Unicode 编码用于表示世界上的各种字符， 中文转换为 \u4e2d\u6587 格式。

HTML 实体编码

将 HTML 特殊字符转义为实体形式， 例如 < 转换为 &lt;。

大小写转换

提供大写、小写、首字母大写、句首大写、大小写反转等功能。

技术原理

哈希算法

MD5 和 SHA1 都是密码学哈希函数，具有以下特性：

  • 单向性：可以从明文计算密文，但无法从密文反推明文
  • 固定长度：无论输入多长，输出长度固定
  • 唯一性：不同的输入理论上会产生不同的输出
  • 雪崩效应：输入微小变化会导致输出显著变化

数据库存储

每次加密操作会自动将 plain_text（明文）、hash_text（密文）、 hash_type（算法类型）存储到云端数据库。 解密时通过密文和算法类型匹配查询数据库返回明文。

技术栈

  • 前端框架：原生 HTML/CSS/JavaScript
  • 样式库：Tailwind CSS
  • 加密库：CryptoJS
  • 数据库：Supabase (PostgreSQL)
