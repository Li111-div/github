# 360°贪吃蛇大作战

一款可在浏览器端运行的360°移动贪吃蛇单人竞技小游戏。

## 运行方式

### 方法一：直接打开（推荐）
双击 `index.html` 文件，使用现代浏览器（Chrome、Firefox、Edge、Safari）打开即可直接运行。

### 方法二：本地服务器
如果遇到资源加载问题，可以使用简单HTTP服务器：

```bash
# Python 3
python -m http.server 8080

# Node.js (需要安装http-server)
npx http-server 8080
```

然后访问 `http://localhost:8080`

## 游戏操作

### PC端
- **移动**：拖动左下角的虚拟摇杆控制方向
- **暂停**：点击右上角暂停按钮

### 移动端
- **移动**：触控屏幕左下角的虚拟摇杆控制方向
- **暂停**：点击右上角暂停按钮

## 游戏模式

### 经典模式
固定速度，无道具，纯长度比拼

### 极速模式
基础速度提升50%，可通过吃道具临时加速

### 道具模式
场内随机生成增益道具：
- ⚡ 加速 - 提升移动速度
- 🐌 减速 - 降低AI蛇速度
- ✂️ 缩短 - 缩短最长AI蛇的长度
- 🛡️ 无敌 - 短时间内免疫碰撞
- 🌱 生长 - 额外增加长度

## 功能特性

### 个性化配置
- **15种蛇皮肤**：像素风、卡通、科技感、渐变色彩等
- **6种背景**：星空、像素草地、赛博朋克、深海、日落、森林
- **音效调节**：背景音乐和音效音量独立调节
- **5档速度**：1x-5x 可调节基础移动速度

### 游戏设置
- **时长选择**：60秒 / 120秒 / 300秒
- **排名系统**：实时显示当前排名
- **结算画面**：显示最终排名、长度、耗时

## 技术说明

- 开发语言：HTML5 + CSS3 + JavaScript
- 渲染引擎：Canvas API
- 兼容性：支持Chrome、Firefox、Safari、Edge及移动端浏览器
- 性能：60FPS流畅运行
