# 无垠扫雷：全域探索

一款超大型网页端扫雷游戏，支持多模式玩法、高度自定义和跨设备流畅运行。

## 🎮 快速开始

### 方式一：双击启动（推荐）
1. 双击 `start.bat` 文件
2. 浏览器会自动打开游戏页面

### 方式二：直接打开
1. 直接在浏览器中打开 `index.html` 文件
2. 部分功能可能受到浏览器安全限制

### 方式三：使用本地服务器
```bash
# 安装 Node.js 后运行
npm install -g http-server
http-server -p 8080
# 访问 http://localhost:8080
```

## 🎯 游戏模式

### 基础模式
- **新手**: 20×15 格，20 个地雷（难度降低 50%）
- **进阶**: 40×25 格，120 个地雷（难度降低 50%）
- **大师**: 60×40 格，400 个地雷（难度降低 50%）

### 特色模式
- **限时挑战**: 30×25 格，100 个地雷，20 分钟倒计时
- **无限扩展**: 20×15 起始地图，8% 雷密度
- **组队协作**: 50×30 格，200 个地雷（本地双人）

## ⌨️ 操作说明

### PC 端
| 操作 | 功能 |
|------|------|
| 左键点击 | 揭开格子 |
| 右键点击 | 标记/取消标记地雷 |
| 双击数字 | 快速开格（周围旗帜数=数字时） |

### 移动端
| 操作 | 功能 |
|------|------|
| 点击 | 揭开格子 |
| 长按 | 标记地雷 |

**注意**: 棋盘为固定式，不可拖拽或缩放。大地图可通过浏览器滚动查看。
