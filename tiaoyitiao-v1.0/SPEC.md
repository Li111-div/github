# 跳跳乐 - 网页小游戏规格文档 (增强版)

## 1. 项目概述

- **项目名称**: 跳跳乐 (Jump Journey)
- **项目类型**: 网页休闲游戏
- **核心功能**: 玩家控制角色从一个平台跳到另一个平台，跳得越远得分越高
- **目标用户**: 手机和电脑用户，休闲玩家

## 2. UI/UX 规格

### 界面系统

1. **主菜单界面** - 游戏启动后的主界面
2. **设置界面** - 难度选择、地图选择
3. **排行榜界面** - 显示各难度/地图的最高分
4. **帮助界面** - 游戏说明和操作指南
5. **游戏界面** - 实际游戏进行画面
6. **结算界面** - 游戏结束后的得分展示

### 布局结构

- **游戏画布**: 全屏自适应，居中显示
- **游戏区域**: 宽度自适应
- **响应式断点**:
  - 手机: < 768px
  - 电脑: >= 768px

### 视觉设计

**沙漠探险**

- 默认: `#FF6B6B` (珊瑚红)

#### 字体

- **得分数字**: 48px (电脑) / 36px (手机)
- **UI文字**: 16px-24px

## 3. 功能规格

### 难度系统

| 难度 | 平台间距 | 平台宽度 | 最大蓄力时间 | 重力 |
|------|----------|----------|--------------|------|
| 简单 | 60-150px | 80-140px | 2.0秒 | 0.4 |
| 中等 | 80-200px | 60-120px | 1.5秒 | 0.6 |
| 困难 | 120-280px | 50-100px | 1.2秒 | 0.8 |

### 地图系统

1. **天空之城** - 默认地图，晴朗天空
2. **森林冒险** - 绿色森林主题
3. **沙漠探险** - 金色沙漠主题
4. **夜晚城市** - 霓虹城市夜景

### 得分规则
- 基础分: 每次成功跳跃 +1 分
- 完美着陆 (中心区域): +2 分
- 连击倍数: 连续成功 3 次起 x2, 5 次起 x3, 以此类推
- 难度加成: 困难 x1.5, 中等 x1.2, 简单 x1.0

### 操作方式

- **手机**: 触摸按住蓄力，松开跳跃
- **电脑**: 鼠标左键按住蓄力，松开跳跃

### 界面切换流程

```
主菜单 → [开始游戏] → 游戏界面 → 结算界面
       → [设置]     → 设置界面 → 主菜单
       → [排行榜]   → 排行榜界面 → 主菜单
       → [帮助]     → 帮助界面 → 主菜单
```

## 4. 验收标准

### 功能检查点
- [ ] 可以选择三种难度进行游戏
- [ ] 可以选择四种地图进行游戏
- [ ] 主菜单、设置、排行榜、帮助、结算界面正常切换
- [ ] 难度和地图设置会影响游戏体验
- [ ] 最高分按难度和地图分别保存

### 视觉检查点
- [ ] 不同地图有不同的视觉风格
- [ ] 界面切换动画流畅
- [ ] 游戏在手机和电脑下正确显示
