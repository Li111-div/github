# Tetris Game - 俄罗斯方块游戏

## 项目概览
这是一个基于原生HTML/CSS/JavaScript开发的经典俄罗斯方块游戏，支持键盘和触屏操作，具备主题切换、丰富的动画效果和音效系统。

### 技术栈
- **前端**: 原生 HTML5 + CSS3 + JavaScript (ES6+)
- **渲染**: Canvas API
- **音效**: Web Audio API (合成音效，无需外部资源)
- **存储**: localStorage (最高分)
- **样式**: 内嵌CSS (像素风格字体、深色主题)

### 核心功能
- 经典俄罗斯方块玩法
- 7种标准方块（I、O、T、S、Z、J、L）
- 键盘控制 + 虚拟按键（触屏支持）
- 20种主题配色方案（每5000分自动切换）
- 计分系统 + 难度递增（速度随等级提升）
- 下一个方块预览
- 暂停/继续功能
- 最高分本地存储

### 动画系统
- 落地动画（缩放+淡出）
- 消除动画（行闪烁+冲击波）
- 粒子系统（消除时发射粒子）
- 屏幕震动（落地/消除时）
- 连击系统（1秒内连续消除显示Combo）
- 闪光效果（消除时屏幕闪光）

### 音效系统（增强版）
使用Web Audio API合成高质量音效，无需外部资源依赖。

#### 音效特性
- **ADSR包络**: 所有音符使用Attack-Decay-Sustain-Release包络，音色更自然
- **谐波叠加**: 主音符叠加泛音，增加音色厚度
- **滤波器**: 使用低通滤波器平滑声音
- **混响效果**: 创建卷积混响，增加空间感

#### 背景音乐（5种风格）
游戏包含5种不同风格的背景音乐，每获得5000分自动切换：

| 序号 | 音乐名称 | 风格 | BPM | 特点 |
|-----|---------|------|-----|------|
| 1 | Classic Tetris | 经典8-bit | 120 | 俄罗斯方块经典主题曲改编，方波旋律+三角波低音 |
| 2 | Neon Dreams | 电子合成器 | 128 | 科技感锯齿波旋律，带琶音装饰，适合快速游戏 |
| 3 | Zen Garden | 轻柔舒缓 | 90 | 柔和正弦波，带Pad垫音，适合放松游戏 |
| 4 | Retro Wave | 8-bit复古 | 140 | 经典8-bit风格，带电子鼓点，节奏感强 |
| 5 | Arcade Hero | 英雄史诗 | 135 | 锯齿波主旋律+三角波和声，激励感强 |

音乐特性：
- 每首音乐4个段落循环播放
- 根据BPM自动计算节拍间隔
- 支持多层音轨（旋律、低音、琶音、和声、Pad、鼓点）
- 暂停时自动停止，恢复时继续播放
- 界面实时显示当前音乐名称

#### 游戏音效
| 音效类型 | 触发时机 | 特点 |
|---------|---------|------|
| 落地音效 | 方块落地 | 低沉有力的"咚"声，带撞击噪声 |
| 消除音效 | 行消除 | 根据行数(1-4)播放不同和弦/琶音 |
| 移动音效 | 左右移动 | 短促轻柔的提示音 |
| 旋转音效 | 方块旋转 | 双音符上升音效 |
| 硬降音效 | 空格键硬降 | 快速下降滑音 + 落地声 |
| 连击音效 | 连续消除 | 音高随连击数递增 |
| 开始音效 | 游戏开始 | 欢快上行琶音 |
| 结束音效 | 游戏结束 | 下行悲伤旋律 |
| 主题切换 | 每5000分 | 上升琶音提示 |
| 点击音效 | 按钮交互 | 清脆的UI反馈音 |

#### 音效控制
- 点击静音按钮(🔊/🔇)切换所有音效
- 静音时自动停止背景音乐

## 构建和运行命令

### 开发环境
```bash
# 静态文件服务器已配置在 .coze 中
# 默认运行在端口 5000
coze dev
```

### 生产环境
```bash
coze build
coze start
```

## 代码风格指南

### 文件结构
```
index.html - 单文件应用，包含所有HTML、CSS和JavaScript代码
```

### JavaScript规范
- 使用 ES6+ 语法
- 函数按功能模块分组（用注释分隔）
- 常量使用大写下划线命名
- 函数使用驼峰命名
- 使用 requestAnimationFrame 实现动画循环

### CSS规范
- 使用 CSS Grid 和 Flexbox 布局
- 动画使用 CSS transitions + keyframes
- 响应式设计（支持移动端）

### 游戏逻辑结构
```
常量定义（方块形状、颜色、配置）
↓
Canvas初始化
↓
游戏状态变量
↓
音效系统
↓
主题系统
↓
动画系统
↓
方块操作（创建、移动、旋转）
↓
碰撞检测
↓
行消除
↓
游戏控制（开始、暂停、结束）
↓
渲染系统
↓
事件监听
↓
主循环
```
