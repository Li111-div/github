# 贪吃蛇360°大对决

一款可在浏览器端运行的360°移动贪吃蛇单人竞技小游戏。

## 运行方式

### 方法一：直接打开（推荐）
双击 `index.html` 文件，使用现代浏览器（Chrome、Firefox、Edge、Safari）打开即可直接运行。

### 方法二：本地服务器
如果遇到资源加载问题，可以使用简单HTTP服务器：

```bash
# Python 3
python -m http.server 8080

# Node.js (需要安装http-server)
npx http-server 8080
```

然后访问 `http://localhost:8080`

## 游戏操作

### PC端
- **移动**：拖动左下角的虚拟摇杆控制方向
- **暂停**：点击右上角暂停按钮

### 移动端
- **移动**：触控屏幕左下角的虚拟摇杆控制方向
- **暂停**：点击右上角暂停按钮

## 游戏模式

### 经典模式
固定速度，无道具，纯长度比拼

### 极速模式
基础速度提升50%，可通过吃道具临时加速

### 道具模式
场内随机生成增益道具：
- ⚡ 加速 - 提升移动速度
- 🐌 减速 - 降低AI蛇速度
- ✂️ 缩短 - 缩短最长AI蛇的长度
- 🛡️ 无敌 - 短时间内免疫碰撞
- 🌱 生长 - 额外增加长度

## 功能特性

### 核心玩法
- **初始配置**：玩家/AI蛇初始长度为50节
- **豆豆系统**：
  - 150个豆豆同时在场
  - 10%概率生成大豆豆（增加8节长度）
  - 生成速度提升至2倍以上（800ms间隔）
- **地图系统**：
  - 4000x4000超大地图
  - 摄像机视角跟随玩家蛇平滑移动
  - 玩家蛇始终处于视野中心±10%范围

### AI智能升级
- **避障系统**：优先躲避边界和其他蛇身
- **追食策略**：主动向豆豆密集区域移动
- **抢食机制**：预判玩家路径并拦截
- **成长策略**：优先寻找价值高的大豆豆
- **差异化行为**：
  - 激进型：主动抢食玩家附近的豆豆
  - 保守型：优先躲避，寻找安全区域
  - 猎手型：专门寻找大豆豆
  - 平衡型：综合策略

### 个性化配置
- **35种皮肤**：
  - 基础风格：像素风、卡通、科技感、渐变彩
  - 赛博朋克：霓虹、赛博粉、赛博蓝、矩阵
  - 国风系列：中国龙、凤凰、翡翠、水墨、樱花
  - 卡通萌系：喵喵、汪汪、兔兔、熊猫、独角兽
  - 元素系列：火焰、冰霜、闪电、大地、风
  - 光影特效：黄金、白银、彩虹、紫罗兰、暗夜、银河、极光
  - 特殊效果：水晶、糖果、自然、熔岩
  - 动态效果：发光、霓虹、火焰、彩虹、星光等

- **6种背景**：星空、像素草地、赛博朋克、深海、日落、森林

- **音效调节**：背景音乐和音效音量独立调节

- **5档速度**：1x-5x 可调节基础移动速度

- 开发语言：HTML5 + CSS3 + JavaScript

- 兼容性：支持Chrome、Firefox、Safari、Edge及移动端浏览器
- 性能：60FPS流畅运行
- 存储方式：LocalStorage

## 游戏规则

### 胜利条件
- 单局时间内蛇的长度最长者获胜
- 排名根据最终长度决定

### 失败条件
- 玩家蛇碰撞边界
- 玩家蛇碰撞其他蛇身

### 时长选择
- 60秒快速局
- 120秒标准局
- 300秒长局

## 注意事项

1. 首次打开可能有轻微加载延迟
2. 建议在网络稳定环境下运行
3. 移动端建议开启竖屏锁定以获得最佳体验
4. 如遇画面显示异常，尝试刷新页面
5. AI蛇碰撞边界会自动重新规划路径
6. 历史记录存储在本地，清除浏览器数据会丢失
